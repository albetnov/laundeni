## Kebutuhan

Untuk menjalankan projek ini. Setidaknya, anda harus memiliki:

-   Apache 2 atau Nginx
-   MySQL
-   PHPMyAdmin

Atau untuk versi bundle:

-   XAMPP
-   Laragon
-   Valet

## Pemasangan

1. Clone repository ini
2. Setting database di .env
3. Ketik php artisan key:generate
4. Ketik php artisan build:laundeni
